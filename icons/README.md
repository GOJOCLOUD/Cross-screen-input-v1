# Windows图标文件

本目录包含Windows软件图标文件，在Mac上完成操作。

## 文件说明

- `windows_icon.ico` - Windows图标文件（包含多种尺寸）
- `round_corner_image_by_atool.png` - 原始圆角图片
- `iconset/` - 包含各种尺寸的PNG图标文件

## 图标尺寸

- 16x16 - 任务栏小图标
- 32x32 - 桌面图标
- 48x48 - 控制面板图标
- 64x64 - 中等大小图标
- 128x128 - 大图标视图
- 256x256 - 超大图标视图

## 生成工具

使用ImageMagick和macOS sips工具生成，在Mac上完成操作。

## 使用方法

将 `windows_icon.ico` 文件设置为Windows应用程序的图标即可。
